# 전후사진 등록

{% hint style="info" %}
## 전후사진이 중요한 이유

전후사진은 시술 효과를 보여주는 **가장 좋은 방법**입니다.

전후사진이 있으면 **예약 전환율이 올라갑니다!**
{% endhint %}

---

## 📍 등록 위치

파트너센터 → 전후사진 → 전후사진 등록


---

## 📝 등록 단계

{% tabs %}
{% tab title="Step 1. 사진 준비" %}
### 사진 규격

| 항목 | 규격 |
|------|:---:|
| 크기 | **800 x 800** 픽셀 이상 |
| 형식 | JPG, PNG |
| 용량 | 5MB 이하 |

{% hint style="danger" %}
### ⚠️ 중요!

**전/후 사진은 같은 조건**으로 촬영해야 합니다.

- 같은 조명
- 같은 각도
- 같은 거리
{% endhint %}
{% endtab %}

{% tab title="Step 2. 동의서 준비" %}
### 환자 동의서

전후사진을 쓰려면 **환자 동의**가 필요합니다.

**동의서에 포함할 내용:**
- 사진 활용 목적 (온라인 광고)
- 활용 범위 (자신있나 앱/웹)
- 환자 서명
{% endtab %}

{% tab title="Step 3. 업로드" %}
### 업로드 방법

1. 시술 카테고리 선택
2. **전** 사진 업로드
3. **후** 사진 업로드
4. 환자 동의서 업로드
5. 시술 정보 입력
{% endtab %}

{% tab title="Step 4. 검수" %}
### 검수 대기

업로드하면 검수를 거칩니다.

| 검수 기간 |
|:---:|
| **1~3일** |
{% endtab %}
{% endtabs %}

---

## 📸 촬영 팁

{% hint style="success" %}
### 좋은 전후사진

| 항목 | 좋은 예 | 나쁜 예 |
|------|:---:|:---:|
| 조명 | 밝고 균일 | 그림자 많음 |
| 배경 | 단색 배경 | 복잡한 배경 |
| 각도 | 매번 동일 | 매번 다름 |
| 보정 | 색보정만 | 과도한 포토샵 |
{% endhint %}
