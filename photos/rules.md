# 전후사진 규정

{% hint style="danger" %}
## 꼭 지켜야 하는 규정!

전후사진은 **의료법 규정**이 있습니다.

지키지 않으면 **반려**되거나 **과태료**를 받을 수 있습니다.
{% endhint %}

---

## ✅ 필수 조건

{% tabs %}
{% tab title="환자 동의" %}
### 환자 동의 필수

| 필수 항목 |
|----------|
| ✅ 환자 서면 동의서 |
| ✅ 온라인 게재 동의 |
{% endtab %}

{% tab title="촬영 조건" %}
### 촬영 조건

| 필수 조건 |
|----------|
| ✅ 전/후 같은 **조명** |
| ✅ 전/후 같은 **각도** |
| ✅ 전/후 같은 **거리** |
{% endtab %}

{% tab title="고지 문구" %}
### 고지 문구 필수

전후사진에는 이 문구가 필요합니다:

시술 결과는 개인에 따라 다를 수 있습니다.

{% endtab %}
{% endtabs %}

---

## ❌ 금지 사항

| 금지 | 내용 |
|:---:|------|
| ❌ | 동의 없는 사진 사용 |
| ❌ | 촬영 조건 조작 (조명, 각도 다르게) |
| ❌ | 과도한 포토샵 보정 |
| ❌ | 타인/타병원 사진 도용 |
| ❌ | 효과 과장 |

---

## 🔴 반려 사유 및 수정 방법

| 반려 사유 | 수정 방법 |
|----------|----------|
| 전/후 조건 다름 | 같은 조건으로 재촬영 |
| 동의서 없음 | 동의서 첨부 |
| 과도한 보정 | 원본에 가깝게 수정 |

---

## ⚠️ 위반 시 제재

| 위반 | 제재 |
|------|------|
| 규정 위반 | 해당 사진 삭제 |
| 반복 위반 | 전후사진 등록 제한 |
| 심각한 위반 | **과태료** (의료법 위반) |
