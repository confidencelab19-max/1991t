# 파트너센터 사용법

{% hint style="info" %}
## 파트너센터란?

병원이 자신있나를 관리하는 **웹사이트**입니다.

이벤트 등록, 예약 관리, 리뷰 답변 등을 여기서 합니다.
{% endhint %}

---

## 🔐 로그인 방법

| 순서 | 방법 |
|:---:|------|
| 1 | **[partner.jasininna.com](https://partner.jasininna.com)** 접속 |
| 2 | 아이디/비밀번호 입력 |
| 3 | 로그인 완료! |

{% hint style="warning" %}
**비밀번호를 잊으셨나요?**

로그인 화면에서 "비밀번호 찾기"를 클릭하세요.
{% endhint %}

---
