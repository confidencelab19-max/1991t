# 입점 신청

{% hint style="info" %}
## 입점 신청은 온라인으로 간단하게!

**3~5일** 안에 심사가 완료됩니다.
{% endhint %}

---

## 📋 필요한 서류

{% hint style="warning" %}
**미리 준비하세요!**

아래 서류가 있어야 신청할 수 있습니다.
{% endhint %}

| 서류 | 설명 | 형식 |
|------|------|:---:|
| **사업자등록증** | 사본 | PDF/사진 |
| **의료기관 개설허가증** | 비뇨의학과 포함 필수 | PDF/사진 |
| **전문의 자격증** | 비뇨의학과 전문의 | PDF/사진 |
| **대표자 신분증** | 주민번호 뒷자리 가림 | PDF/사진 |

---

## 🚀 신청 단계

{% tabs %}
{% tab title="Step 1" %}
### 사이트 접속

**[partner.jasininna.com](https://partner.jasininna.com)** 에 접속하세요.

"**입점 신청**" 버튼을 클릭합니다.
{% endtab %}

{% tab title="Step 2" %}
### 본인 인증

휴대폰 본인 인증을 진행합니다.
{% endtab %}

{% tab title="Step 3" %}
### 정보 입력

병원 정보를 입력합니다.

- 병원명
- 대표자명
- 사업자등록번호
- 병원 주소
- 연락처
{% endtab %}

{% tab title="Step 4" %}
### 서류 업로드

준비한 서류를 업로드합니다.

> 💡 파일 크기는 개당 **10MB 이하**로 해주세요.
{% endtab %}

{% tab title="Step 5" %}
### 심사 대기

신청 완료! **3~5일** 안에 결과를 알려드려요.

결과는 **문자 + 이메일**로 안내됩니다.
{% endtab %}
{% endtabs %}

---

## ❓ 자주 묻는 질문

{% tabs %}
{% tab title="비용" %}
### Q. 입점하는 데 비용이 드나요?

**아니요, 입점은 무료입니다.**

예약이 발생했을 때만 수수료(10%)가 발생해요.
{% endtab %}

{% tab title="자격" %}
### Q. 개인 병원도 입점 가능한가요?

**네, 가능합니다.**

규모와 상관없이 비뇨의학과 전문의가 있으면 됩니다.
{% endtab %}

{% tab title="심사" %}
### Q. 심사에서 떨어지면요?

탈락 사유를 알려드립니다.

보완 후 **30일 뒤에 다시 신청**할 수 있어요.
{% endtab %}

{% tab title="노출" %}
### Q. 입점하면 바로 앱에 나오나요?

**아니요.**

입점 후 병원 정보와 이벤트를 등록해야 노출됩니다.

👉 [병원 정보 등록하기](../hospital/info.md)
{% endtab %}
{% endtabs %}
