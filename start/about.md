# 서비스 소개

{% hint style="info" %}
## 자신있나란?

**비뇨기과 전문 플랫폼**입니다.

남성 고객들이 비뇨기과 시술 정보를 찾고, 병원을 예약하는 앱이에요.
{% endhint %}

## ✅ 병원에 좋은 점

{% hint style="success" %}
### 1. 타겟 고객만 모여있어요

비뇨기과 시술에 관심 있는 고객만 있어서 **전환율이 높아요.**
{% endhint %}

{% hint style="success" %}
### 2. 예약 관리가 편해요

앱으로 예약이 들어와서 **전화 응대가 줄어요.**
{% endhint %}

{% hint style="success" %}
### 3. 홍보가 돼요

이벤트만 등록하면 **자동으로 노출**돼요.
{% endhint %}

{% hint style="success" %}
### 4. 리뷰 관리가 쉬워요

고객 리뷰를 **한 곳에서 관리**할 수 있어요.
{% endhint %}
