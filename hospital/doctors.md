# 의사 정보 등록

{% hint style="info" %}
## 의사 정보가 중요한 이유

고객은 **어떤 의사가 시술하는지** 궁금해합니다.

의사 정보가 잘 등록되어 있으면 신뢰도가 올라갑니다!
{% endhint %}

---

## 📍 설정 위치

파트너센터 → 병원 관리 → 의사 정보 → 의사 추가


---

## 📝 등록할 정보

{% tabs %}
{% tab title="기본 정보" %}
### 필수 입력 항목

| 항목 | 예시 |
|------|------|
| **이름** | 홍길동 |
| **직함** | 대표원장 |
| **전문의 자격** | 비뇨의학과 전문의 |
{% endtab %}

{% tab title="프로필 사진" %}
### 프로필 사진 규격

| 항목 | 규격 |
|------|:---:|
| 크기 | **400 x 500** 픽셀 이상 |
| 복장 | 정장 또는 흰 가운 |
| 배경 | 깔끔한 단색 |

{% hint style="warning" %}
**팁:** 밝은 표정의 정면 사진이 좋아요!
{% endhint %}
{% endtab %}

{% tab title="학력/경력" %}
### 학력 및 경력 (선택)

서울대학교 의과대학 졸업
서울대병원 비뇨의학과 전공의 수료
비뇨의학과 전문의 취득 (2015)
대한비뇨의학회 정회원

{% hint style="info" %}
**팁:** 경력이 많을수록 신뢰도가 올라갑니다!
{% endhint %}
{% endtab %}
{% endtabs %}

### ❌ 이렇게 쓰면 안 돼요!

| 잘못된 표기 | 이유 |
|------------|------|
| 비뇨기과 명의 | 공인되지 않은 표현 |
| 남성수술 전문의 | 그런 자격증 없음 |
| 국내 최고 전문의 | 의료법 위반 |
{% endhint %}
