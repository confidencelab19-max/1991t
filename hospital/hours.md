# 운영 시간 설정

{% hint style="info" %}
## 정확한 운영 시간 설정이 중요해요!

잘못된 시간이 등록되면 고객 불만이 생깁니다.
{% endhint %}

---

## 📍 설정 위치

파트너센터 → 병원 관리 → 운영 시간
---

## ⏰ 진료 시간 설정

### 요일별 설정 예시

| 요일 | 진료 시간 | 점심 시간 |
|:---:|:---:|:---:|
| 월~금 | 09:00 - 18:00 | 12:00 - 13:00 |
| 토요일 | 09:00 - 14:00 | 없음 |
| 일요일 | **휴진** | - |

---

## 📅 휴진일 설정

{% tabs %}
{% tab title="공휴일" %}
### 공휴일 휴진 설정

"**공휴일 자동 휴진**" 체크하면 공휴일에 자동으로 휴진 처리됩니다.
{% endtab %}

{% tab title="특정 날짜" %}
### 특정 날짜 휴진

달력에서 휴진할 날짜를 직접 선택합니다.

예: 병원 워크샵, 개인 휴가 등
{% endtab %}

{% tab title="정기 휴진" %}
### 정기 휴진

매주 반복되는 휴진을 설정합니다.

예: 매주 수요일 오후 휴진
{% endtab %}
{% endtabs %}

{% hint style="warning" %}
### ⚠️ 중요!

**휴진일은 최소 7일 전에 설정하세요.**

이미 예약이 잡힌 날짜는 휴진 설정이 어렵습니다.
{% endhint %}

---

## 🕐 예약 가능 시간

예약을 받을 시간대를 설정해주세요.

| 설정 | 권장값 |
|------|:---:|
| 예약 시작 | 진료 시작 30분 후 (예: 09:30) |
| 예약 마감 | 진료 종료 1시간 전 (예: 17:00) |
| 점심시간 | 예약 차단 |
