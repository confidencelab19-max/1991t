# 병원 정보 설정

{% hint style="info" %}
## 병원 정보가 중요한 이유

병원 정보가 잘 등록되어 있어야 **고객이 신뢰**합니다.

빈 칸 없이 꼼꼼하게 채워주세요!
{% endhint %}

---

## 📍 설정 위치
파트너센터 → 병원 관리 → 기본 정보
---

## 📝 등록할 정보

{% tabs %}
{% tab title="1. 병원 소개글" %}
### 병원 소개글 작성

병원을 소개하는 글을 작성해주세요.

#### ✅ 좋은 예시

자신있나비뇨의학과는 2015년 개원 이래 남성 건강 전문 의료기관으로 10년간 운영해왔습니다.

비뇨의학과 전문의 3인이 상주하며, 정관수술, 포경수술, 남성확대술을 전문으로 합니다.

최신 장비와 철저한 사후 관리로 높은 만족도를 유지하고 있습니다.


#### ❌ 나쁜 예시 (의료법 위반)

국내 최고의 비뇨기과! 100% 만족 보장!


{% hint style="danger" %}
**주의!** "최고", "100%" 같은 표현은 **의료법 위반**입니다.

👉 [쓰면 안 되는 표현 보기](../medical-law/prohibited.md)
{% endhint %}
{% endtab %}

{% tab title="2. 병원 사진" %}
### 병원 사진 등록

고객이 병원을 미리 볼 수 있도록 사진을 올려주세요.

#### 필수 사진

| 종류 | 설명 |
|------|------|
| **외관 사진** | 병원 간판이 보이는 전경 |
| **내부 사진** | 대기실, 상담실, 진료실 |

#### 사진 규격

| 항목 | 규격 |
|------|:---:|
| 크기 | **1200 x 800** 픽셀 이상 |
| 형식 | JPG 또는 PNG |
| 용량 | 5MB 이하 |

{% hint style="warning" %}
**팁:** 밝고 깨끗한 사진이 좋습니다!
{% endhint %}
{% endtab %}

{% tab title="3. 주소/연락처" %}
### 주소 입력

| 항목 | 예시 |
|------|------|
| **주소** | 서울시 강남구 테헤란로 123 OO빌딩 5층 |
| **지하철** | 2호선 강남역 3번 출구 도보 5분 |
| **주차** | 건물 내 지하주차장 (2시간 무료) |

### 연락처 입력

| 항목 | 예시 |
|------|------|
| **대표 전화** | 02-1234-5678 |
| **상담 전화** | 010-1234-5678 (선택) |
{% endtab %}
