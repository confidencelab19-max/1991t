## 1. 개요 및 취지

{% hint style="info" %}
**근거:** 의료법 제56조(의료광고의 금지 등), 제57조(의료광고의 심의)
{% endhint %}

본 플랫폼은 의료광고심의 매체\*가 아닌 플랫폼에서도 모든 파트너 병원이 **합법적인 의료 이벤트를 운영**하실 수 있도록 돕기 위해 이 가이드를 만들었습니다.

> \*의료광고심의 매체 = 전년도 말 기준 직전 3개월간 일일 평균 사용자 수 10만 명 이상의 인터넷 매체
> (출처: 의료광고심의위원회)

---

### 플랫폼의 역할과 책임 구분

| 역할 | 설명 |
|:---|:---|
| **원장님(병원)** | 광고의 주체예요. 내용을 만들고 등록하시는 분이에요 |
| **저희 플랫폼** | 원장님의 광고가 환자분께 잘 전달되도록 돕는 정보중개 매체예요 |

{% hint style="warning" %}
**광고 내용의 적법성에 대한 최종 책임은 등록하시는 병원(의료인등)에 있어요.**

저희는 형식 검수와 명백한 위반 콘텐츠 필터링을 통해 원장님을 보호해 드립니다.

**이 가이드를 따라주시면 가장 확실한 보호막이 됩니다!**
{% endhint %}
