# 📸 치료경험담 · 전후사진

{% hint style="info" %}
**근거:** 의료법 제56조 제2항 제2호, 시행령 제23조, 표시·광고의 공정화에 관한 법률 제3조
{% endhint %}

---

## 6-1. 치료경험담 — 원칙적 금지

{% hint style="danger" %}
인터넷 상 불특정 다수에게 환자의 치료경험담·치료후기 형태로 표현되는 광고는 **의료법 제56조 제2항 제2호 위반**이에요.
{% endhint %}

### 금지되는 유형

| 유형 |
|:---|
| 환자의 직접 치료경험담을 인용한 광고 |
| 의료인의 환자 치료 사례를 인용한 광고 |
| 특정인 이름이 들어간 후기 |
| 가상 인물을 실존 인물처럼 기술한 내용 |
| 초상화 형식의 수술 전·후 이미지 (사진에 준하는 효과) |

---

### 대가성 후기 — 절대 금지

{% hint style="danger" %}
환자에게 경제적 이익(할인, 현금, 포인트 등)을 제공하고 치료경험담을 작성하게 하는 것은 **의료법 + 표시광고법 동시 위반**이에요.

**"후기 써주시면 ○○ 드려요"** ← 이 한마디가 이중 위반의 시작입니다.
{% endhint %}

---

## 6-2. 전후사진 — 5가지 요건 모두 충족 시 가능

전후사진은 예외적으로 아래 요건을 **전부 충족**하면 사용 가능해요.

| 번호 | 요건 |
|:---:|:---|
| ① | 해당 의료기관에서 진료한 환자의 사진일 것 |
| ② | 전·후 사진의 인물이 **동일인**일 것 |
| ③ | 전·후의 **촬영 시기를 명시**할 것 |
| ④ | **동일 조건**에서 촬영할 것 (별도의 보정·조작 금지) |
| ⑤ | 해당 진료별로 **부작용을 명시**할 것 |

{% hint style="warning" %}
**+ 환자로부터 사진 사용에 대한 사전 동의 필수!**
{% endhint %}

### 위반 예시

> 수술 후 사진만 조명을 밝게 하거나, 보정하거나, 다른 환경에서 촬영한 경우
> → **의료법 위반 + 표시광고법상 기만적 표시·광고 해당**

## 6-3. 비뇨기과 이미지 특별 규정

| 유형 | 가능 여부 |
|:---|:---:|
| 성기 실사진(노출) | 🚫 절대 금지 |
| 수술 중 절개·봉합·출혈 장면 | 🚫 금지 |
| 보정·조작된 전후사진 | 🚫 금지 |
| 의학 일러스트(해부학적 도해) — 시술 원리 설명 목적 | ✅ 가능 |
| 병원 시설·장비 사진 | ✅ 가능 |

{% hint style="info" %}
의학 일러스트는 **선정적이지 않게** 사용해 주세요.
{% endhint %}
