# ⚖️ 비교 · 비방 광고 금지

{% hint style="info" %}
**근거:** 의료법 제56조 제2항 제4호·제5호
{% endhint %}

---

## 7-1. 이런 건 전부 안 돼요

| 금지 유형 |
|:---|
| 특정 의료인·의료기관과 비교하여 우수하다는 내용 |
| 비교 대상을 특정하지 않더라도, 특정 진료방법이 우월한 것으로 인식되는 광고 |
| 다른 의료인·의료기관을 비방하는 광고 |

---

### 비뇨기과 위반 예시

| ❌ NG |
|:---|
| "○○수술 아무데서나 받지 마세요. 안전하고 효과적인 결과를 약속합니다" |
| "다른 병원에서 실패한 재수술도 가능합니다" (다른 병원 비방) |
| "일반의가 아닌 전문의에게만 맡기세요" (비전문의 비방) |
| "우리 병원만의 독자적 기술 vs 일반적 수술" (우월성 암시) |

---

## 7-2. 이런 건 가능해요

| ✅ OK |
|:---|
| 특정 병원·의료인 언급 없이 일반적 진료방법의 장단점을 객관적으로 나열 |
| 과거 우리 병원 vs 현재 우리 병원 비교 |
| 동일 의료기기의 새 버전과 기존 버전 비교 (개선점 표시) |
| "전문의에게 진료받으시는 것을 권합니다" (비방 표현 없이 권고만) |
