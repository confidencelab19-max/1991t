## 5-2. 0% / 100% 의미를 내포한 표현 — 원칙적 금지

### ❌ 금지

| | | | | |
|:---:|:---:|:---:|:---:|:---:|
| 부작용없이 | 통증 없이 | 완성 | 완치 | 완비 |
| 완벽 | 해결(솔루션) | 가장 안전한 | 안전한 | 만들다 |
| 만들어줍니다 | 무사고 | | | |

### ✅ 허용 가능

> "개선 가능", "도움이 될 수 있다" 등 **가능성을 나타내는 표현**은 맥락에 따라 OK
