## 12. 부작용 고지 의무

{% hint style="info" %}
**근거:** 의료법 제56조 제2항 제7호
{% endhint %}

{% hint style="danger" %}
**모든 시술 이벤트에 부작용 고지는 필수예요!**

부작용을 안 쓰는 것 자체가 위반입니다.
{% endhint %}

### 비뇨기과용 부작용 고지 템플릿

{% hint style="warning" %}
**📋 시술 전 꼭 확인해 주세요**

본 시술은 개인에 따라 출혈, 감염, 부종, 통증, 감각 변화 등이 발생할 수 있습니다. 결과에는 개인차가 있으며, 시술 전 의료진 상담을 통해 충분한 설명을 받으실 수 있습니다.
{% endhint %}

> **배치:** 시술 설명 본문 또는 광고 이미지 하단에 소비자가 쉽게 인지할 수 있는 위치에 넣어주세요.
>
> 팝업 링크로만 안내하는 건 ⚠️ 비권장이에요.

## 13. 사전심의 안내

{% hint style="info" %}
**근거:** 의료법 제57조
{% endhint %}

### 13-1. 사전심의 대상

인터넷을 포함한 특정 매체로 의료광고를 하려면 자율심의기구(대한의사협회 의료광고심의위원회 등)의 **사전심의를 받아야** 해요.

---

### 13-2. 심의 면제 (기본 정보만 담은 광고)

아래 내용**만** 포함하면 사전심의 면제예요 (제57조 제3항).

| 심의 면제 항목 |
|:---|
| 병원 이름 · 주소 · 전화번호 |
| 진료과목 |
| 소속 의료인 이름 · 성별 · 면허 종류 |

{% hint style="warning" %}
**하지만!** 시술 내용, 가격, 전후사진, 이벤트 등이 **하나라도 들어가면 심의 대상**이에요.
{% endhint %}

---

### 13-3. 유효기간

> **승인일로부터 3년**
>
> 만료 후 계속 광고하시려면 **만료 6개월 전 재신청** (제57조 제8항·제9항)

---

### 13-4. 심의 미이행 시 처벌

| 처벌 | 행정처분 |
|:---|:---|
| 1년 이하 징역 또는 1천만원 이하 벌금 (제89조) | 1회 경고 → 2회 업무정지 15일 → 3회 업무정지 1개월 |

{% hint style="success" %}
**💡 심의 번호가 있으시면 이벤트 등록 시 입력해 주세요.**

심의 대상 여부가 헷갈리시면 대한의사협회 의료광고심의위원회(www.admedical.org)에서 확인하실 수 있어요.
{% endhint %}

## 14. 처벌 · 행정처분 한눈에

{% hint style="danger" %}
이걸 보시면 왜 가이드를 지켜야 하는지 바로 느끼실 거예요.
{% endhint %}

### 14-1. 형사처벌

| 위반 유형 | 근거 | 처벌 |
|:---|:---|:---|
| 환자유인·알선·사주 | 제88조 (제27조③ 위반) | **3년 이하 징역 / 3천만원 이하 벌금** |
| 비의료인 의료광고 | 제89조 (제56조① 위반) | 1년 이하 징역 / 1천만원 이하 벌금 |
| 금지 유형 광고 (15가지) | 제89조 (제56조② 위반) | 1년 이하 징역 / 1천만원 이하 벌금 |
| 거짓·과장 광고 | 제89조 (제56조③ 위반) | 1년 이하 징역 / 1천만원 이하 벌금 |
| 사전심의 미이행 | 제89조 (제57조 위반) | 1년 이하 징역 / 1천만원 이하 벌금 |

---

### 14-2. 행정처분

| 위반 유형 | 1회 | 2회 | 3회 |
|:---|:---|:---|:---|
| 거짓 광고 | 업무정지 2개월 | 업무정지 3개월 | 개설허가 취소 |
| 과장 광고 | 업무정지 1개월 | 업무정지 2개월 | 업무정지 3개월 |
| 사전심의 미이행 | 경고 | 업무정지 15일 | 업무정지 1개월 |
| 환자유인 | 자격정지 2개월 | 자격정지 3개월 | 면허취소 가능 |

---

### 14-3. 표시광고법 제재 (별도 적용)

> **공정거래위원회:** 시정명령 + 과징금(매출액의 2% 이내)

---

### 14-4. 플랫폼 자체 조치

| 조치 |
|:---|
| 위반 콘텐츠 비공개 · 삭제 |
| 수정 요청 및 수정 완료 전 게시 중단 |
| 반복 위반 시 파트너 계약 해지 |
