# 🆘 문제 발생 시 대응 가이드

{% hint style="warning" %}
혹시라도 행정기관 조사, 민원, 신고 등이 발생하면 아래 순서로 대응해 주세요.
{% endhint %}

---

## 18-1. 대응 순서

### 1단계 — 조사 통지 확인

> 조사 대상·범위·근거 법령을 확인하고, **통지서 원본을 보관**해 주세요.

### 2단계 — 증빙 자료 즉시 정리

| 준비할 자료 |
|:---|
| 해당 광고 원본 (스크린샷) |
| 사전심의 번호·결과문 |
| 전후사진 환자 동의서 |
| 플랫폼 이용약관·이용료 납부 내역 |

### 3단계 — 플랫폼에 연락

> 저희 파트너센터에 바로 연락해 주세요. **최대한 도움 드리겠습니다.**

### 4단계 — 전문가 조력

> 상황에 따라 **의료 전문 변호사**의 조력을 받으시면 더욱 확실합니다.

---

## 18-2. 자료 보관 — 5년 보관 권장

| 보관 대상 | 기간 | 이유 |
|:---|:---:|:---|
| 등록 광고 원본 (스크린샷) | 5년 | 의료법 위반 공소시효 5년 |
| 사전심의 결과·승인 문서 | 5년 | 심의 이행 입증 |
| 전후사진 환자 동의서 | 5년 | 개인정보·적법성 입증 |
| 이용료 납부 내역 | 5년 | 광고 대가 구조 입증 (환자유인 아님 입증) |
| 가격·할인 이벤트의 정상가 근거 | 5년 | 할인 광고 적법성 입증 |
