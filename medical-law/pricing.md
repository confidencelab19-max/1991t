# 💰 가격 · 할인 · 보험 · 제휴 표기

---

## 9. 가격 · 할인 표기

{% hint style="info" %}
**근거:** 의료법 제56조 제2항 제13호, 시행령 제23조 제1항 제13호
{% endhint %}

### 9-1. 비급여 가격 공개 — OK!

> 비급여 진료비 정보 공개는 소비자 알 권리 차원에서 **허용**되고 있어요. 안심하고 표기하셔도 돼요.

---

### 9-2. 할인 광고 시 필수 표기 사항

비급여 진료비용 할인·면제 광고에는 **반드시 아래 정보가 포함**되어야 해요.

| 필수 항목 | 설명 |
|:---|:---|
| 할인·면제 금액 | 실제 할인 금액 또는 면제 범위 |
| 할인 대상 | 누구에게 적용되는지 |
| 할인 기간 | 시작일 ~ 종료일 (정확한 일자) |
| 할인 범위 | 어떤 시술에 적용되는지 |
| 할인 이전의 비급여 진료비용 | 실제 정상가 (부풀리면 안 돼요!) |
| 의료기관 유지운영에 필요한 비용 충당 가능 여부 | 지나치게 낮은 가격 설정 시 유인 판단 가능 |

---

### 9-3. 묶음상품 가격

여러 시술을 종합한 묶음상품은 **개별 시술의 정상가/할인가를 각각 명시**해야 하고, 고객이 개별 시술을 선택할 수 있어야 해요.

| ❌ NG | ✅ OK |
|:---|:---|
| "확대+조루 = 300만원" (개별가 미표기) | 개별 시술 가격 각각 표기 + 묶음가 + 차이 이유 설명 |
| "1회 7만원, 5회 30만원" (차이 설명 없음) | 1회 가격 + 5회 묶음가 + "묶음 할인 ○만원 적용" 표기 |

---

### 9-4. 시즌·이벤트 표현

| ❌ NG | ✅ OK |
|:---|:---|
| "겨울한정", "크리스마스 한정" | "이벤트 기간: 12월 1일 ~ 12월 31일" (정확한 일자) |
| "10월 특별 이벤트 10명 선착순" | 정확한 기간 + 대상 + 조건 명시 |
| "선착순 무료 시술" (조건 없음) | 대상·기간·수량·조건 구체 명시 |

---

### 9-5. 기타 금지

| 금지 항목 |
|:---|
| 사진/후기 제공 조건 할인 ❌ (뒷광고 + 치료경험담 위반) |
| "사진 제공 및 후기 작성 조건 없음" 표시도 ❌ |
| "친구와 함께 방문 시 할인" ❌ (제3자유인) |
| "VIP 패키지", "프리미엄 검진" ❌ (환자유인 소지) |
| 차량 제공 표현 ❌ (환자유인) |
| 경품 제공 암시 ❌ |

## 10. 보험 관련 표기

{% hint style="info" %}
**근거:** 의료법 제56조 제2항 제13호, 보건복지부 가이드라인(2판)
{% endhint %}

### 10-1. 실손보험(실비보험) 표기

{% hint style="warning" %}
"실손보험", "실비보험" 문구는 **상세이미지 또는 상세페이지 안에서만** 가능하고, **대표 이미지·카드 이미지에는 표기 불가**예요.
{% endhint %}

**반드시 아래 문구를 전부 삽입해야 해요:**

| 번호 | 필수 문구 |
|:---:|:---|
| ① | "미용 목적 시술/수술에는 실손보험 적용이 불가합니다" 또는 "치료 목적 시술/수술만 실손보험 적용이 가능합니다" |
| ② | "해당 병원에서만 적용 가능한 것은 아닙니다" |
| ③ | "의료소비자가 가입한 해당 실손보험 가입 조건에 따라 적용 여부 및 적용 범위가 다를 수 있습니다" |

| ❌ NG | ✅ OK |
|:---|:---|
| 대표 이미지에 "실비보험 적용!" | 상세페이지에서 위 3개 문구와 함께 안내 |
| "보장범위 80~90% 가능" | 위 필수 문구 삽입 후 상세페이지에서만 안내 |

---

### 10-2. 건강보험 표기

비급여 진료에 "건강보험" 문구는 **원칙적으로 사용 불가**해요. 비급여인데 건강보험이 되는 것으로 오인시킬 수 있거든요.

**예외:** 실제로 건강보험 적용이 가능한 케이스이고, 상세이미지/상세페이지에 다음을 모두 기재한 경우

| 번호 | 필수 기재 |
|:---:|:---|
| ① | 보험 적용 기준 명시 (예: 진단코드 및 진단명) |
| ② | "보험 적용 가능 여부는 진단 후 확인이 가능하며, 수술 방법에 따라 자기부담금이 달라질 수 있습니다" |

{% hint style="warning" %}
허용되는 경우라도 건강보험 혜택으로 **부가 광고(홍보)하는 것은 안 돼요.**

비뇨기과 예시: 포경수술이 건강보험 적용이 되는 경우라도, "건강보험으로 저렴하게!" 같은 홍보성 문구는 ❌
{% endhint %}
