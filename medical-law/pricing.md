# 가격 표시 규정

{% hint style="info" %}
## 가격은 투명하고 정확하게!

잘못된 가격 표시는 **반려** 또는 **제재** 대상입니다.
{% endhint %}

---

## ✅ 올바른 가격 표시

{% tabs %}
{% tab title="기본 규칙" %}
### 기본 규칙

| 규칙 | 설명 |
|------|------|
| **VAT 포함** | 모든 가격은 VAT 포함 총액 |
| **실제 정상가** | 정상가는 실제로 받는 가격 |
| **추가 비용 명시** | 마취비 등 별도 비용 표시 |
{% endtab %}

{% tab title="좋은 예시" %}
### 올바른 가격 표시

정관수술 이벤트

정상가: 500,000원 이벤트가: 350,000원

※ 마취비 별도: 50,000원 ※ 모든 가격은 VAT 포함입니다.

{% endtab %}

{% tab title="나쁜 예시" %}
### 잘못된 가격 표시

정관수술 90% 할인!

정상가: 3,500,000원 ← 뻥튀기 가격! 이벤트가: 350,000원


{% hint style="danger" %}
원래 50만원인데 정상가를 350만원으로 표시하면 **허위 광고**입니다.
{% endhint %}
{% endtab %}
{% endtabs %}

---

## ❌ 금지 사항

| 금지 | 설명 |
|:---:|------|
| ❌ | **허위 정상가** - 원래 안 받던 가격을 정상가로 표시 |
| ❌ | **숨은 비용** - 추가 비용을 숨기고 나중에 청구 |
| ❌ | **미끼 가격** - 저렴하게 유인 후 다른 시술 권유 |
| ❌ | **원가 이하 판매** - 불법 (환자유인행위) |

---

## 📝 추가 비용 표시 방법

별도 비용이 있다면 **꼭 표시**하세요.

| 항목 | 표시 예시 |
|------|----------|
| 마취비 | 마취비 별도: 50,000원 |
| 약값 | 약 처방 시 별도 |
| 검사비 | 사전 검사 별도: 30,000원 |

## ✅ 가격 입력 체크리스트

{% hint style="success" %}
### 가격 입력 전에 확인하세요!

- [ ] **VAT 포함** 금액인가?
- [ ] 정상가가 **실제로 받는 가격**인가?
- [ ] **추가 비용**을 명시했는가?
- [ ] 할인율이 적절한가? (80% 이상은 의심받음)
{% endhint %}
