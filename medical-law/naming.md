# 🏥 의료기관 명칭 · 진료과목 표기

## 2. 의료기관 명칭 표기

{% hint style="info" %}
**근거:** 의료법 제3조(의료기관), 제42조(의료기관 명칭), 의료법 시행규칙 제40조
{% endhint %}

> 병원 이름은 **의료기관개설신고증명서 / 개설허가증**에 있는 명칭을 그대로 써주세요.

---

### 기본 원칙

| 원칙 | 설명 |
|:---|:---|
| 한글 표시 | 의료기관 명칭은 **한글 표시가 원칙**이에요. 외국어가 포함된 경우 외국어 철자 또는 한글과 병기해 주세요. |
| 클리닉 사용 | 의원급에서 **'클리닉' 또는 'clinic'**을 '의원'과 함께 병기하는 것은 ✅ 가능해요. |
| 센터 사용 | **'센터' 또는 'center'**는 의원·병원급에서 ❌ 사용 불가, **종합병원급에서만** 사용 가능해요. |

---

### ⚠️ 전문병원 표기 — 특히 주의!

| 상황 | 가능 여부 |
|:---|:---:|
| 보건복지부 장관이 지정한 전문병원 → "○○전문병원" | ✅ OK |
| 지정받지 않았는데 "전문병원" 표시 | ❌ NG |
| 전문병원이지만 지정받지 않은 과목에 "전문병원" 표시 | ❌ NG |
| "전문병원"이 아니어도 "○○ 전문" 용어를 병원명과 함께 사용 | ❌ NG |

---

### 비뇨기과 예시

| ❌ NG | ✅ OK |
|:---|:---|
| "남성전문병원 ○○비뇨의학과" | "○○비뇨의학과의원" |
| "비뇨기 전문 ○○의원" | "○○비뇨의학과의원 (비뇨의학과 전문의 ○○○)" |

## 3. 진료과목 표기

{% hint style="info" %}
**근거:** 의료법 제43조 제5항, 의료법 시행규칙 제41조
{% endhint %}

의료기관이 표시할 수 있는 진료과목은 **의료법에서 정한 과목으로 제한**돼요.

---

### 병원·의원 진료과목 (26개)

내과, 신경과, 정신건강의학과, 외과, 정형외과, 신경외과, 흉부외과, 성형외과, 마취통증의학과, 산부인과, 소아청소년과, 안과, 이비인후과, 피부과, **비뇨의학과**, 영상의학과, 방사선종양학과, 병리과, 진단검사의학과, 재활의학과, 결핵과, 가정의학과, 핵의학과, 직업환경의학과, 응급의학과, 예방의학과

---

### 주의 사항

| 규칙 | 예시 |
|:---|:---|
| 세부진료과목 병기 **불가** | "비뇨의학과 남성의학클리닉" ❌ |
| 질병명 + clinic **가능** | "전립선 클리닉" ✅ |
