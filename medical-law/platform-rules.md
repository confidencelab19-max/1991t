# 📋 플랫폼 이벤트 운영 규칙

> **Part 2. 플랫폼 이벤트 운영 규칙**

---

## 15. 이벤트 등록 기본 규칙

### 15-1. 제목 규칙

| 규칙 |
|:---|
| 시술명 또는 이벤트 내용이 **명확하게** 드러나야 해요 |
| 제목과 내용이 **일치**해야 해요 (단, 치료 방향성이 동일하면 허용) |
| 제목에 **금지 표현**(최고, 유일, 전문 등)은 사용 불가 |

---

### 15-2. 이미지 규격

| 항목 | 기준 |
|:---|:---|
| 대표 이미지 | 플랫폼 등록 가이드 별도 안내 |
| 상세 이미지 | 플랫폼 등록 가이드 별도 안내 |
| **금지 이미지** | 성기 실사진, 수술 장면, 선정적 이미지, 조작 사진 |
| **허용 이미지** | 의학 일러스트(도해), 병원·장비 사진 |

---

### 15-3. 중복 등록 금지

> 동일 시술 · 동일 조건의 이벤트를 **중복 등록하실 수 없어요.**

---

### 15-4. 수정 시 주의

{% hint style="warning" %}
이벤트 내용을 수정하신 후 재게시하시면, 사전심의 대상인 경우 **재심의가 필요**할 수 있어요.

심의 내용과 다른 광고는 **제56조 제2항 제11호 위반**이에요.
{% endhint %}

---

## 16. 미성년자 대상 이벤트

{% hint style="info" %}
**근거:** 민법 제5조(미성년자의 능력)
{% endhint %}

미성년자(만 18세 미만)를 대상으로 하는 이벤트는 아래 문구를 **상세이미지 또는 상세페이지에 반드시 삽입**해 주세요.

| 번호 | 필수 문구 |
|:---:|:---|
| ① | **대상자 표기:** "○○을 원하는 만 ○세 이상 ○세 미만의 시술/수술 희망자" |
| ② | **미성년자 안내:** "미성년자의 시술/수술을 원할 경우 법정대리인(부모님)의 동의하에 진행이 가능합니다" |
