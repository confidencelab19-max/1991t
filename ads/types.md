# 광고 종류

{% hint style="info" %}
## 광고란?

광고를 하면 우리 병원 이벤트가 **더 많이 노출**됩니다.
{% endhint %}

{% hint style="warning" %}
**광고 안 해도 노출은 됩니다!**

하지만 광고하면 **상위에 노출**돼서 더 많은 고객이 봅니다.
{% endhint %}

---

## 📊 광고 종류 비교

| 광고 | 과금 방식 | 최소 비용 | 특징 |
|------|:---:|:---:|------|
| **CPC** | 클릭당 | 100원~ | 클릭한 만큼만 비용 |
| **배너** | 기간제 | 50만원/주 | 메인 화면 노출 |
| **푸시** | 건당 | 협의 | 푸시 알림 발송 |

---

## 💰 광고별 상세 안내

{% tabs %}
{% tab title="CPC 광고" %}
### CPC 광고 (클릭당 과금)

고객이 이벤트를 **클릭할 때만** 비용이 발생합니다.

| 항목 | 내용 |
|------|------|
| 과금 시점 | 클릭할 때 |
| 최소 금액 | 클릭당 100원 |
| 추천 금액 | 클릭당 300~500원 |

**이런 병원에 추천:**
- 예약 전환율이 높은 병원
- 적은 비용으로 시작하고 싶은 병원
{% endtab %}

{% tab title="배너 광고" %}
### 배너 광고 (기간제)

앱 **메인 화면**에 큰 배너로 노출됩니다.

| 항목 | 내용 |
|------|------|
| 과금 방식 | 1주/2주/1개월 |
| 최소 비용 | 주 50만원~ |
| 노출 위치 | 앱 메인 상단 |

**이런 병원에 추천:**
- 병원 인지도를 높이고 싶은 병원
- 신규 오픈 병원
{% endtab %}

{% tab title="푸시 광고" %}
### 푸시 광고 (건당)

고객 휴대폰에 **푸시 알림**으로 보냅니다.

| 항목 | 내용 |
|------|------|
| 과금 방식 | 발송 건당 |
| 비용 | 협의 |
| 노출 위치 | 휴대폰 알림 |

**이런 병원에 추천:**
- 특가 이벤트를 알리고 싶은 병원
- 빠른 예약 유도가 필요한 병원
{% endtab %}
{% endtabs %}

## 🎯 어떤 광고를 선택할까요?

| 목적 | 추천 광고 |
|------|:---:|
| 적은 비용으로 시작 | **CPC** |
| 병원 인지도 높이기 | **배너** |
| 특가 이벤트 알리기 | **푸시** |

{% hint style="success" %}
**처음이라면 CPC 광고를 추천해요!**

적은 비용으로 시작할 수 있습니다.
{% endhint %}
