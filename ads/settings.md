# 광고 설정

{% hint style="info" %}
## CPC 광고 설정 방법

가장 많이 사용하는 CPC 광고 설정 방법을 알려드립니다.
{% endhint %}

---

## 📍 설정 위치

파트너센터 → 광고 관리 → 광고 설정

---

## ⚙️ 설정 단계

{% tabs %}
{% tab title="Step 1" %}
### 광고할 이벤트 선택

광고하고 싶은 이벤트를 선택합니다.
{% endtab %}

{% tab title="Step 2" %}
### 클릭당 금액 설정

| 금액 | 효과 |
|:---:|------|
| 100~200원 | 기본 노출 |
| **300~500원** | 상위 노출 (추천) |
| 500원 이상 | 최상위 노출 |

{% hint style="info" %}
금액이 높을수록 상위에 노출됩니다.

하지만 너무 높이면 비용이 많이 나갑니다.
{% endhint %}
{% endtab %}

{% tab title="Step 3" %}
### 일일 예산 설정

하루에 쓸 **최대 금액**을 정합니다.

| 예산 | 예상 클릭 (300원 기준) |
|:---:|:---:|
| 일 3만원 | 약 100회 |
| 일 5만원 | 약 166회 |
| 일 10만원 | 약 333회 |

예산을 다 쓰면 그날은 광고가 멈춥니다.
{% endtab %}

{% tab title="Step 4" %}
### 광고 시작

**"광고 시작"** 버튼을 클릭하면 바로 광고가 시작됩니다.
{% endtab %}
{% endtabs %}

---

## 💡 권장 설정 (처음 시작할 때)

{% hint style="success" %}
### 추천 설정

| 항목 | 설정값 |
|------|:---:|
| 클릭당 금액 | **300원** |
| 일일 예산 | **5만원** |
{% endhint %}

---

## ⏸️ 광고 중지/재개

언제든 광고를 **중지**하거나 **다시 시작**할 수 있습니다.

파트너센터 → 광고 관리 → 광고 목록 → 중지/재개
