# 광고 효과 보기

{% hint style="info" %}
## 광고가 잘 되고 있는지 확인하세요!

데이터를 보고 광고를 개선할 수 있습니다.
{% endhint %}

---

## 📍 확인 위치

파트너센터 → 광고 관리 → 광고 성과


---

## 📊 봐야 할 숫자들

| 지표 | 의미 | 좋은 기준 |
|------|------|:---:|
| **노출수** | 광고가 보여진 횟수 | 많을수록 좋음 |
| **클릭수** | 광고를 클릭한 횟수 | 많을수록 좋음 |
| **클릭률** | 노출 대비 클릭 비율 | **3% 이상** |
| **전환수** | 예약까지 이어진 횟수 | 많을수록 좋음 |
| **광고비** | 총 쓴 비용 | - |

---

## 📈 클릭률 계산법

클릭률 = 클릭수 ÷ 노출수 × 100


| 클릭률 | 평가 |
|:---:|:---:|
| 3% 이상 | ⭐⭐⭐ 좋음 |
| 1~3% | ⭐⭐ 보통 |
| 1% 미만 | ⭐ 개선 필요 |

---

## 🔧 광고 개선 방법

{% tabs %}
{% tab title="클릭률이 낮으면" %}
### 클릭률 개선 방법

| 원인 | 해결 방법 |
|------|----------|
| 이미지가 안 끌림 | 이미지 변경 |
| 가격이 안 끌림 | 가격 조정 |
| 제목이 안 끌림 | 제목 변경 |
{% endtab %}

{% tab title="전환이 안 되면" %}
### 전환율 개선 방법

| 원인 | 해결 방법 |
|------|----------|
| 설명이 부족 | 시술 설명 보강 |
| 리뷰가 없음 | 리뷰 요청 |
| 전후사진 없음 | 전후사진 등록 |
{% endtab %}
{% endtabs %}

---

## 💰 예산 조정

{% hint style="success" %}
**효과가 좋은 광고:**
- 예산 늘리기
- 클릭당 금액 올리기
{% endhint %}

{% hint style="warning" %}
**효과가 나쁜 광고:**
- 예산 줄이기
- 광고 내용 수정 후 다시 시작
{% endhint %}
