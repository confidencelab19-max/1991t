# 별점 관리

{% hint style="info" %}
## 별점이 중요한 이유

별점이 높으면:
- 검색에서 **상위 노출**
- 고객 **신뢰도 상승**
- **예약 전환율 상승**
{% endhint %}

---

## 📊 별점 확인 위치

파트너센터 → 리뷰 관리 → 리뷰 통계


| 확인 항목 |
|----------|
| 평균 별점 |
| 이번 달 별점 |
| 월별 별점 추이 |

## ⭐ 별점 올리는 방법

{% tabs %}
{% tab title="서비스 품질" %}
### 서비스 품질 향상

| 항목 | 방법 |
|------|------|
| 시술 결과 | 전문성 유지 |
| 친절도 | 전 직원 CS 교육 |
| 대기 시간 | 예약 시간 준수 |
| 시설 | 청결하고 쾌적하게 |
{% endtab %}

{% tab title="리뷰 요청" %}
### 리뷰 요청하기

만족한 고객에게 리뷰를 부탁하세요.

시술 결과에 만족하셨다면 자신있나 앱에서 리뷰 남겨주시면 감사하겠습니다!

{% endtab %}

{% tab title="답변 달기" %}
### 모든 리뷰에 답변

답변을 다는 것만으로도 인상이 좋아집니다.

👉 [리뷰 답변 방법 보기](reply.md)

### 허위/악성 리뷰인 경우

2. 플랫폼에 신고

파트너센터 → 리뷰 관리 → 해당 리뷰 → 신고


| 신고 사유 |
|----------|
| 허위 사실 |
| 방문 이력 없음 |
| 악의적 비방 |
