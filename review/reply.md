# 리뷰 답변

{% hint style="info" %}
## 리뷰 답변이 중요한 이유

리뷰 답변은 **다른 고객도 봅니다.**

정성스러운 답변은 병원 신뢰도를 높입니다!
{% endhint %}

---

## 📍 리뷰 확인 위치

파트너센터 → 리뷰 관리 → 리뷰 목록


새 리뷰가 오면 **알림**이 옵니다.

---

## ✍️ 답변 방법

1. 답변할 리뷰 클릭
2. 답변 작성
3. **"답변 등록"** 클릭

---

## 💬 별점별 답변 예시

{% tabs %}
{% tab title="⭐⭐⭐⭐⭐ 5점" %}
### 좋은 리뷰 답변

OOO님, 따뜻한 후기 감사합니다!

시술 결과에 만족하셨다니 저희도 기쁩니다. 앞으로도 좋은 서비스로 보답하겠습니다.

건강하세요!

{% endtab %}

{% tab title="⭐⭐⭐ 3점" %}
### 보통 리뷰 답변

OOO님, 솔직한 후기 감사합니다.

말씀해주신 부분 확인하고 개선하겠습니다. 다음에 방문하시면 더 나은 서비스로 보답하겠습니다.

감사합니다.

{% endtab %}

{% tab title="⭐ 1~2점" %}
### 아쉬운 리뷰 답변

OOO님, 불편을 드려 죄송합니다.

말씀해주신 내용 진심으로 받아들이겠습니다. 자세한 상담을 원하시면 02-1234-5678로 연락 주세요. 성심껏 도와드리겠습니다.

다시 한번 사과드립니다.

{% endtab %}
{% endtabs %}

## ⚠️ 답변 시 주의사항

{% hint style="danger" %}
### ❌ 하지 마세요!

- 고객과 싸우기
- 변명만 하기
- 답변 안 하기
- 복사/붙여넣기만 하기
{% endhint %}

{% hint style="success" %}
### ✅ 이렇게 하세요!

- **24시간 내** 답변
- 진심 어린 감사/사과
- 리뷰 내용에 맞는 **맞춤 답변**
- 개선 의지 표현
{% endhint %}
