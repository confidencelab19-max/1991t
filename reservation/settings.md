# 예약 설정

{% hint style="info" %}
## 예약 설정이란?

고객이 앱에서 예약할 때의 **규칙을 정하는 것**입니다.
{% endhint %}

---

## 📍 설정 위치

파트너센터 → 예약 관리 → 예약 설정


---

## ⚙️ 설정 항목

{% tabs %}
{% tab title="1. 확정 방식" %}
### 예약 확정 방식 선택

| 방식 | 설명 | 추천 |
|------|------|:---:|
| **자동 확정** | 예약 즉시 자동 확정 | ⭐ |
| 수동 확정 | 직접 확정/거절 선택 | - |

{% hint style="success" %}
### 💡 자동 확정을 추천해요!

수동 확정은 **24시간 내** 처리 안 하면 자동 취소됩니다.
{% endhint %}
{% endtab %}

{% tab title="2. 예약 시간" %}
### 예약 가능 시간 설정

| 설정 | 예시 |
|------|:---:|
| 예약 시작 시간 | 09:30 |
| 예약 마감 시간 | 17:00 |
| 시간 단위 | 30분 |

{% hint style="info" %}
점심시간은 자동으로 예약이 차단됩니다.
{% endhint %}
{% endtab %}

{% tab title="3. 동시 예약" %}
### 동시 예약 제한

같은 시간에 받을 수 있는 최대 예약 수입니다.

| 의사 수 | 권장 설정 |
|:---:|:---:|
| 1명 | 시간당 2건 |
| 2명 | 시간당 4건 |
| 3명+ | 시간당 6건 |
{% endtab %}

{% tab title="4. 예약 마감" %}
### 예약 마감 기준

| 설정 | 의미 |
|------|------|
| **당일 예약 가능** | 오늘 예약도 받음 |
| 1일 전 마감 | 내일부터 예약 가능 |
| 2일 전 마감 | 모레부터 예약 가능 |
{% endtab %}
{% endtabs %}

---

## 📋 권장 설정 예시

{% hint style="success" %}
### 일반적인 병원 설정

| 항목 | 설정값 |
|------|:---:|
| 예약 확정 | 자동 확정 |
| 예약 시간 | 09:30 ~ 17:00 |
| 시간 단위 | 30분 |
| 동시 예약 | 시간당 3건 |
| 예약 마감 | 당일 예약 가능 |
{% endhint %}
