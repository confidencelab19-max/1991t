# 노쇼 관리

{% hint style="warning" %}
## 노쇼란?

예약 후 **연락 없이 안 오는 고객**입니다.

노쇼가 많으면 병원 운영에 손실이 생깁니다.
{% endhint %}

---

## 🛡️ 노쇼 예방 방법

{% tabs %}
{% tab title="자동 알림" %}
### 리마인드 알림

자신있나에서 자동으로 알림을 보냅니다.

| 시점 | 알림 내용 |
|------|----------|
| **1일 전** | "내일 예약이 있습니다" |
| **당일 아침** | "오늘 예약이 있습니다" |

{% hint style="info" %}
알림은 **기본으로 켜져 있어요.**

별도 설정 필요 없습니다.
{% endhint %}
{% endtab %}

{% tab title="확인 전화" %}
### 예약 확인 전화

중요한 예약은 직접 전화로 확인하세요.

**통화 예시:**
"안녕하세요, OOO병원입니다. 내일 예약 확인차 연락드렸습니다. 예정대로 방문 가능하신가요?"
**확인 전화 권장 대상:**
- 고가 시술 예약
- 첫 방문 고객
- 노쇼 이력 있는 고객
{% endtab %}
{% endtabs %}

---

## 📝 노쇼 처리 방법

예약 시간이 지났는데 안 오면:

1. 예약 목록에서 해당 예약 클릭
2. **"노쇼 처리"** 버튼 클릭
3. 완료

---

## ⚠️ 노쇼 고객 관리

자신있나에서 노쇼 이력을 관리합니다.

| 노쇼 횟수 | 조치 |
|:---:|------|
| 1회 | 기록만 |
| 2회 | 예약 시 경고 표시 |
| 3회+ | 예약 제한 (7일) |

{% hint style="info" %}
**노쇼 이력이 있는 고객은 예약 목록에 표시됩니다.**

예약 확인 전화를 꼭 해주세요!
{% endhint %}
