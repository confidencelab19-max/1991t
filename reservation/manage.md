# 예약 관리

{% hint style="info" %}
## 예약이 들어오면?

알림이 오고, 파트너센터에서 확인할 수 있어요.
{% endhint %}

---

## 🔔 예약 알림

예약이 들어오면 알림을 받습니다.

| 알림 방법 |
|:---:|
| 앱 푸시 알림 |
| 이메일 |
| 파트너센터 알림 |

---

## 📋 예약 목록 확인

파트너센터 → 예약 관리 → 예약 현황


### 예약 상태

| 상태 | 의미 |
|:---:|------|
| 🟡 **대기** | 확정 필요 (수동 확정인 경우) |
| 🟢 **확정** | 예약 완료 |
| ⚪ **취소** | 예약 취소됨 |
| 🔵 **완료** | 방문 완료 |
| 🔴 **노쇼** | 방문 안 함 |

---

## ✅ 예약 처리하기

{% tabs %}
{% tab title="예약 확정" %}
### 예약 확정하기 (수동 확정인 경우)

1. 예약 목록에서 **"대기"** 상태 예약 클릭
2. 예약 정보 확인
3. **"예약 확정"** 클릭

{% hint style="warning" %}
**24시간 내 처리하세요!**

안 하면 자동 취소됩니다.
{% endhint %}
{% endtab %}

{% tab title="예약 변경" %}
### 예약 변경하기

고객이 변경 요청하면:

1. 해당 예약 클릭
2. **"예약 변경"** 클릭
3. 새로운 날짜/시간 선택
4. 저장

고객에게 자동으로 알림이 갑니다.
{% endtab %}

{% tab title="예약 취소" %}
### 예약 취소하기

부득이하게 취소해야 할 때:

1. 해당 예약 클릭
2. **"예약 취소"** 클릭
3. 취소 사유 입력

{% hint style="danger" %}
**병원에서 취소하면 고객 불만이 생길 수 있어요.**

가능하면 다른 시간을 제안해주세요.
{% endhint %}
{% endtab %}
{% endtabs %}
