# 상담 팁

{% hint style="success" %}
## 좋은 상담 = 높은 예약률

상담을 잘 하면 예약 전환율이 올라갑니다!
{% endhint %}

---

## ⏱️ 응답 속도

| 응답 시간 | 평가 |
|:---:|:---:|
| **5분 이내** | ⭐⭐⭐ 최고 |
| 30분 이내 | ⭐⭐ 좋음 |
| 1시간 이내 | ⭐ 보통 |
| 1시간 이상 | 고객이 떠날 수 있음 |

{% hint style="warning" %}
**빠른 응답이 중요해요!**

고객은 여러 병원에 동시에 문의합니다.
{% endhint %}
