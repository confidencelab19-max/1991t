# 채팅 상담

{% hint style="info" %}
## 채팅 상담이란?

고객이 앱에서 병원에 **채팅으로 문의**하는 기능입니다.
{% endhint %}

---

## 📍 채팅 확인 위치

파트너센터 → 상담 관리 → 채팅


새 채팅이 오면 **알림**이 옵니다.

---

## ⚙️ 자동응답 설정

상담원이 바로 답변 못 할 때 **자동으로 보내는 메시지**입니다.

파트너센터 → 상담 관리 → 자동응답 설정


{% tabs %}
{% tab title="영업시간 내" %}
### 영업시간 내 자동응답

안녕하세요! OOO병원입니다. 상담원이 곧 답변드리겠습니다. 잠시만 기다려주세요.

{% endtab %}

{% tab title="영업시간 외" %}
### 영업시간 외 자동응답

안녕하세요! OOO병원입니다. 현재 영업시간이 아닙니다.

영업시간: 평일 09:00-18:00

남겨주신 문의는 다음 영업일에 답변드리겠습니다.

{% endtab %}
{% endtabs %}

---

## 🔔 알림 설정

파트너센터 → 상담 관리 → 알림 설정


| 설정 | 권장 |
|------|:---:|
| 푸시 알림 | ✅ ON |
| 소리 알림 | ✅ ON |
| 이메일 알림 | ✅ ON |

{% hint style="success" %}
**모든 알림을 켜두세요!**

새 채팅을 놓치지 않습니다.
{% endhint %}

---

## 💾 자주 쓰는 문구 저장

자주 쓰는 답변을 미리 저장해두면 빠르게 답변할 수 있어요.

파트너센터 → 상담 관리 → 자주 쓰는 문구


### 저장 예시

| 문구 이름 | 내용 |
|----------|------|
| 인사 | 안녕하세요! OOO병원입니다. 무엇을 도와드릴까요? |
| 예약 안내 | 예약을 도와드릴게요. 원하시는 날짜와 시간을 말씀해주세요. |
| 가격 안내 | 해당 시술의 이벤트 가격은 OO만원입니다. |
| 마무리 | 더 궁금하신 점 있으시면 말씀해주세요! |
