# 정산 안내

{% hint style="info" %}
## 정산이란?

예약이 발생하면 **수수료를 제외한 금액**을 정산받습니다.
{% endhint %}

---

## 💰 수수료

| 항목 | 수수료 |
|------|:---:|
| 앱 예약 수수료 | 예약 금액의 **10%** |

### 계산 예시

| 항목 | 금액 |
|------|-----:|
| 시술 가격 | 500,000원 |
| 수수료 (10%) | -50,000원 |
| **정산 금액** | **450,000원** |

---

## 📅 정산 일정

| 항목 | 날짜 |
|------|------|
| 정산 기간 | 매월 1일 ~ 말일 |
| 정산 확정 | 다음 달 **5일** |
| 입금일 | 다음 달 **15일** |

### 예시

1월 예약 → 2월 5일 정산 확정 → 2월 15일 입금
---

## 📊 정산 내역 확인

파트너센터 → 정산 → 정산 내역


| 확인 가능 항목 |
|---------------|
| 총 예약 금액 |
| 총 수수료 |
| 정산 금액 |
| 예약별 상세 내역 |

---

## 🧾 세금계산서

| 항목 | 내용 |
|------|------|
| 발행 시기 | 매월 초 (전월분) |
| 발행 방식 | 이메일 전송 |
| 확인 방법 | 파트너센터 → 정산 → 세금계산서 |
