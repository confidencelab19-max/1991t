# 이벤트 등록

{% hint style="info" %}
## 이벤트란?

**앱에 노출되는 시술 상품**입니다.

이벤트를 등록해야 고객이 우리 병원을 볼 수 있어요!
{% endhint %}

---

## 📍 등록 위치

파트너센터 → 이벤트 관리 → 이벤트 등록


---

## 📝 등록 단계

{% tabs %}
{% tab title="Step 1. 시술 선택" %}
### 시술 카테고리 선택

| 선택 | 예시 |
|------|------|
| **대분류** | 정관수술 |
| **세부 시술** | 무도정관수술 |

👉 [전체 시술 목록 보기](../procedures/categories.md)
{% endtab %}

{% tab title="Step 2. 이벤트명" %}
### 이벤트 이름 작성

고객에게 보여지는 제목입니다.

#### ✅ 좋은 예시

| 예시 |
|------|
| 정관수술 이벤트 |
| [신규 고객] 정관수술 특가 |
| 10년 경력 전문의 정관수술 |

#### ❌ 나쁜 예시 (의료법 위반)

| 예시 | 문제점 |
|------|--------|
| 국내 최저가 정관수술 | "최저가" 금지 |
| 100% 만족 보장! | "100%" 금지 |
| 연예인이 선택한 병원 | 유명인 언급 금지 |

{% hint style="danger" %}
**위 표현을 쓰면 반려됩니다!**

👉 [금지 표현 전체 보기](../medical-law/prohibited.md)
{% endhint %}
{% endtab %}

{% tab title="Step 3. 가격" %}
### 가격 입력

| 항목 | 입력값 |
|------|:---:|
| **정상가** | 500,000원 |
| **이벤트가** | 350,000원 |

{% hint style="warning" %}
### ⚠️ 가격 입력 규칙

1. **VAT 포함** 총액으로 입력
2. **정상가**는 실제로 받는 가격
3. 추가 비용(마취비 등)은 **설명에 명시**
{% endhint %}
{% endtab %}

{% tab title="Step 4. 설명" %}
### 시술 설명 작성

고객이 궁금해하는 정보를 적어주세요.

[시술 설명] 무도정관수술은 절개 없이 진행하는 정관수술입니다.

[시술 시간] 약 15-20분

[마취 방법] 국소마취

[회복 기간] 2-3일 후 일상생활 가능

[주의사항]

시술 후 2주간 금주
시술 후 1주일간 격렬한 운동 자제
[추가 비용]

마취비 별도: 50,000원
{% endtab %}

{% tab title="Step 5. 이미지" %}
### 이벤트 이미지 등록

| 항목 | 규격 |
|------|:---:|
| 크기 | **720 x 720** 픽셀 |
| 형식 | JPG, PNG |
| 용량 | 5MB 이하 |

{% hint style="danger" %}
**이미지에 금지 표현이 있으면 반려돼요!**

"최저가", "100% 만족" 같은 문구를 넣지 마세요.
{% endhint %}
{% endtab %}

{% tab title="Step 6. 완료" %}
### 검수 요청

모든 정보를 입력하고 **"검수 요청"** 버튼을 클릭하세요.

| 항목 | 내용 |
|------|------|
| 검수 기간 | **1~2일** (영업일 기준) |
| 결과 안내 | 알림으로 안내 |

{% hint style="info" %}
검수를 통과하면 앱에 바로 노출됩니다!
{% endhint %}
{% endtab %}
{% endtabs %}
