# 이벤트 규칙

{% hint style="warning" %}
## 규칙을 지켜주세요!

규칙을 어기면 이벤트가 중지되거나 제재를 받을 수 있습니다.
{% endhint %}

---

## 💰 가격 규칙

{% tabs %}
{% tab title="✅ 지켜야 할 것" %}
### 필수 규칙

| 규칙 | 설명 |
|------|------|
| **VAT 포함** | 모든 가격은 VAT 포함 총액 |
| **실제 정상가** | 정상가는 실제로 받는 가격 |
| **추가 비용 명시** | 마취비 등 추가 비용 표시 |
{% endtab %}

{% tab title="❌ 하면 안 되는 것" %}
### 금지 사항

| 금지 | 예시 |
|------|------|
| **허위 정상가** | 원래 50만원인데 정상가를 200만원으로 표시 |
| **숨은 비용** | 마취비를 안 알려주고 나중에 청구 |
| **원가 이하 판매** | 불법 (환자유인행위) |
{% endtab %}
{% endtabs %}

## 📅 이벤트 기간 규칙

| 항목 | 기준 |
|------|:---:|
| 최소 기간 | 7일 |
| 최대 기간 | 90일 |
| 연장 | 종료 전에 새로 등록 |

## ⚠️ 규칙 위반 시 제재

| 횟수 | 조치 |
|:---:|------|
| 1회 | 경고 |
| 2회 | 해당 이벤트 중지 (7일) |
| 3회 | 신규 이벤트 등록 제한 (30일) |
| 4회+ | 파트너 자격 정지 검토 |
