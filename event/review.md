# 이벤트 검수

{% hint style="info" %}
## 검수란?

등록한 이벤트가 **의료법을 지키고 있는지** 확인하는 과정입니다.

검수를 통과해야 앱에 노출돼요.
{% endhint %}

---

## ⏱️ 검수 과정

| 단계 | 내용 | 소요 시간 |
|:---:|------|:---:|
| 1 | 자동 검사 (필수 항목 체크) | 즉시 |
| 2 | 담당자 검토 | 1~2일 |
| 3 | 승인 또는 반려 | - |

---

## 📋 검수 결과 확인

파트너센터 → 이벤트 관리 → 이벤트 목록
| 상태 | 의미 |
|:---:|------|
| 🟡 **검수 대기** | 검수 중이에요 |
| 🟢 **승인** | 앱에 노출 중! |
| 🔴 **반려** | 수정이 필요해요 |

---

## 🔴 반려되면 어떻게 하나요?

{% tabs %}
{% tab title="1. 사유 확인" %}
### 반려 사유 확인

반려된 이벤트를 클릭하면 **반려 사유**가 나옵니다.
{% endtab %}

{% tab title="2. 수정하기" %}
### 반려 사유별 수정 방법

#### 금지 표현 사용

| 반려 내용 | 수정 방법 |
|----------|----------|
| "국내 최고" 표현 사용 | 해당 표현 삭제 |

**수정 예시:**
| 수정 전 | 수정 후 |
|--------|--------|
| 국내 최고의 정관수술 | 10년 경력 전문의의 정관수술 |

#### 이미지 내 금지 표현

| 반려 내용 | 수정 방법 |
|----------|----------|
| 이미지에 "최저가" 포함 | 새 이미지로 교체 |

#### 가격 정보 오류

| 반려 내용 | 수정 방법 |
|----------|----------|
| 이벤트가가 정상가보다 높음 | 가격 다시 확인 |
{% endtab %}

{% tab title="3. 재검수" %}
### 재검수 요청

수정을 완료하고 **"재검수 요청"** 버튼을 클릭하세요.

재검수는 **당일 또는 다음날** 처리됩니다.
{% endtab %}
{% endtabs %}

## ✅ 반려를 줄이는 방법

{% hint style="success" %}
### 등록 전 체크리스트

- [ ] 금지 표현 없음 (최고, 100%, 유일 등)
- [ ] 이미지에 문제 문구 없음
- [ ] 가격이 정확함 (이벤트가 < 정상가)
- [ ] VAT 포함 가격임
- [ ] 추가 비용 명시함
{% endhint %}

👉 [금지 표현 전체 목록 보기](../medical-law/prohibited.md)
